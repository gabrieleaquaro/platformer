# Platformer builder 

Project created for the Computer Graphics course at PoliMi.

Deployed  [here](https://gabrieleaquaro.github.io/platformer/).


## Objective 
Build a street to the golden goal at the top. 

## Commands 

#### Playmode: 

Move : WASD

Jump : Spacebar

Enter builder mode: K

#### Builder mode:

Zoom : wheel

Move camera: wasd

Place block: GUI
